以2014到2019`台指期連續月`的日k棒為分析對象

自訂11個k棒漲跌排列的特徵

用`隨機森林演算法`算出哪幾個特徵影響績效比較顯著

當特徵出現的`隔天`

採取`開盤買進、收盤賣出`的日內交易策略

手續費、稅、滑價…等交易成本設為4點

畫出每個策略的各自累計損益

以及將停損考慮進去後的優化績效
